# Tract It

A clean, modern and fast personal finance tracker built with Flutter, designed to help users monitor daily expenses and income with simplicity and clarity.
This project includes real-time UI updates, chart insights, offline storage using Hive, and beautiful lightweight animations.

## Getting Started

Features

 Secure Summary Visibility
 Toggle visibility for Total Balance, Total Income, and Total Expense with one switch.

Keeps sensitive amounts hidden when needed.

 Add, Edit & Delete Transactions

Supports adding both expenses and income from the same form and model.

Edit existing items without creating duplicates.

Delete a single entry or clear all transactions.

 State Management (BLoC)

Uses the flutter_bloc architecture for clean business logic.

Real-time UI synchronization for all cards and expense tiles.

Local Persistent Storage

Powered by Hive (lightweight NoSQL database).

Fast local reads, writes, updates, and deletes.

Items sorted by date automatically.

 Insightful Analytics

Beautiful spending chart that updates instantly when new income or expenses are added.

 Clean UI + Theming

Custom color palette.

Light/Dark mode support.

Smooth SpeedDial for quick Add Expense / Add Income actions.
