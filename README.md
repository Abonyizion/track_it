Track It is a clean and modern Flutter application designed to help users manage expenses, track income, and maintain financial clarity with an intuitive UI.
Built with scalable architecture, smooth navigation, and state-management best practices.

# Getting Started

## Tech Stack & Architecture
- Flutter Clean Architecture
- Feature-First Folder Structure
- Repository Pattern
- BLoC State Management
- Local Storage: Hive

## Features

### Onboarding
- PageView onboarding flow
  - Swipe navigation
  - Next/back buttons

### Dashboard
- Displays:
  - Total Balance
  - Total Income
  - Total Expenses
- Clean financial summary cards

### Expense & Income Management
- Add expenses using a floating action button (FAB)
  - Dual popup options: Add Income / Add Expense
- Dynamic form fields with validation
- View list of all transactions
- Clear all expenses (built-in functionality)

## UI/UX
- Dark/Light mode ready
- Reusable widgets (cards, inputs, buttons, dialogs)
